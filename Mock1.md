\- \*\*Variables and Assignment:\*\*

&nbsp; - Variables can be assigned single or multiple values.

&nbsp; - Single value assignment example: `a = 10`

&nbsp; - Multiple value assignment example: `a, b, c = 10, 20, 30`

&nbsp; - Syntax errors occur if the number of variables and values mismatch (e.g., assigning three values to two variables).

&nbsp; - The \*\*`id()` function\*\* returns the memory address of variables.

&nbsp; - Variables with the same value generally have \*\*different identities\*\*, but for some immutable types like small integers, identities may be the same (candidate’s explanation was partially unclear).



\- \*\*Operators in Python:\*\*

&nbsp; - Types mentioned: Arithmetic, Logical, Assignment, Comparison (relational) operators.

&nbsp; - Candidate was more confident explaining the \*\*arithmetic operator\*\* for power: `\*\*`.

&nbsp; - Less confident about relational operators.



\- \*\*Conditional Constructs:\*\*

&nbsp; - Constructs cited: `if`, `else`, and nested `if` statements.

&nbsp; - Difference between `if` and `else` explained as:

&nbsp;   - `if` for single condition.

&nbsp;   - `else` executes when `if` condition is false.

&nbsp; - Example: Checking if a number is even using modulus operator (`%`).

&nbsp; - Indentation is critical for block definition in Python.

&nbsp;   - Mixing indentation levels causes errors.



\- \*\*Loops:\*\*

&nbsp; - Purpose: To repeat statements multiple times.

&nbsp; - Types in Python: `for` loop and `while` loop.

&nbsp; - Differences:

&nbsp;   - `for` loop is used when the number of iterations is known (often with `range()`).

&nbsp;   - `while` loop is used when the number of iterations is unknown, controlled by a condition.

&nbsp; - Both single and multiple statements can be repeated in loops.



\- \*\*Functions:\*\*

&nbsp; - Defined as blocks of code.

&nbsp; - Types:

&nbsp;   - User-defined functions (created by programmers).

&nbsp;   - Built-in functions (predefined in Python).

&nbsp;   - Lambda functions (anonymous functions).

&nbsp; - Examples of built-in functions mentioned: `min()`, `max()`, `sum()`, etc.

&nbsp; - User-defined function syntax begins with `def`.



\- \*\*String Functions:\*\*

&nbsp; - Examples explained:

&nbsp;   - `count()` – counts occurrences of a substring or character.

&nbsp;   - `len()` – returns the length of the string.

&nbsp;   - Difference: `count()` counts specific substring occurrences; `len()` returns total length.

&nbsp; - Strip functions:

&nbsp;   - `lstrip()` removes spaces from the left side.

&nbsp;   - `rstrip()` removes spaces from the right side.

&nbsp;   - `strip()` removes spaces from both sides.
